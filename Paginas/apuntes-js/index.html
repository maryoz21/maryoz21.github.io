Flujo formulario — el esqueleto completo
const form = document.getElementById("formRegistro");
form.addEventListener("submit", (e) => {
  e.preventDefault(); // ← SIEMPRE, evita recargar

  let input = document.getElementById("miInput");
  let span  = document.getElementById("miError");
  let regex = /^[A-Za-z]+$/;
  let todoCorrecto = true;

  if (regex.test(input.value)) {
    marcarOk(input, span);
  } else {
    marcarError(input, span, "mensaje");
    todoCorrecto = false;
  }

  if (todoCorrecto) { /* éxito */ }
  else { /* hay errores */ }
});

Patrón storage — los 3 pasos
// 1. Leer
let items = JSON.parse(
  sessionStorage.getItem("clave")
) || [];

// 2. Modificar
items.push({ dato: valor });

// 3. Guardar
sessionStorage.setItem("clave", JSON.stringify(items));
Las 4 regex de tu código
/^[A-Za-z]+$/Solo letras, mín. 1, sin espacios
/^[A-Za-z0-9._]+@[A-Za-z0-9]+\.[a-zA-Z]{2,}$/Email válido (ejemplo@gmail.com)
/^\d{9}$/Exactamente 9 dígitos (tel.)
/^.{6,}$/Mín. 6 caracteres cualesquiera
Significado de cada símbolo
^Inicio de cadena
$Final de cadena (sin $ acepta basura al final)
+1 o más veces
*0 o más veces
{9}Exactamente 9 veces
{2,}2 o más veces
\dCualquier dígito 0-9
.Cualquier carácter (menos salto de línea)
[A-Za-z]Letra mayúscula o minúscula
\.Punto literal (escapado con \)
Cómo se usa
let regex = /^[A-Za-z]+$/;
regex.test("Mario")   // → true
regex.test("Mario3")  // → false  (tiene número)
regex.test("")        // → false  (+ requiere ≥1)
.test() devuelve true o false — es un booleano

localStorage vs sessionStorage
localStorage Persiste siempre, aunque cierres el navegador
sessionStorage Solo dura la sesión, se borra al cerrar la pestaña
La API es idéntica en ambos — solo cambia el nombre
Los 4 métodos esenciales
// Guardar (siempre stringify para arrays/objetos)
sessionStorage.setItem("clave", JSON.stringify(datos));

// Leer (parse + fallback con || [])
let datos = JSON.parse(
  sessionStorage.getItem("clave")
) || [];

// Borrar una clave
sessionStorage.removeItem("clave");

// Borrar todo
sessionStorage.clear();
Cargar lista al iniciar la página
let tareas = JSON.parse(

tareas.forEach((tarea) => {
  let li = document.createElement("li");
  li.textContent = tarea.tarea; // ← propiedad del objeto
  document.getElementById("listaTareas")
    .appendChild(li);
});
Errores clásicos de examen
Error Olvidar e.preventDefault() — la página recarga y pierdes todo
Error Olvidar $ al final de la regex — /^\d{9}/ acepta "123456789abc"
Error No usar JSON.parse al leer — obtienes string en vez de array
Error No usar JSON.stringify al guardar — guarda "[object Object]"
Error Olvidar || [] — JSON.parse(null) devuelve null y .push() peta
Error No limpiar el input — olvidar input.value = "" tras guardar
Trampa Punto en regex sin escapar — \. es punto literal, . es cualquier carácter
Trampa Dejar clase vieja — siempre hacer .add() y .remove() juntos
